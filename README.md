# Keywords

Python Keywords are **Predefined**

### False, True

- Boolean Data Types

```python
False == (1 > 2) ,True == (2 > 1)
```

### None 

- `Empty` Value Constant

```python
def func():
  x = 2

func() == None 
--------------
True
```

### and, or, not  

Logical Operators

1. x and y : both x and y must be True.
2. x or y : either x or y must be True.
3. not x : x must be False.

```python
x, y = True, False

print(x and y)
print(x or y)
print(not x )

------------------
False
True
False
```
